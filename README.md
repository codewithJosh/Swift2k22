# SWIFT 2K22
Online Reservation/ Ticketing System for Bus Transportation in Lingayen-Dagupan

## FEATURES:sparkles:
- Login and Register
- Create Reservation (Admin only)
- Buy Ticket through Swift Wallet
- Generate Ticket Barcode
